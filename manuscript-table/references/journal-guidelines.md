## Universal Principles

- **Standalone utility**: Title + table + footnotes = complete understanding without narrative text.
- **No vertical lines**: Use only 3 horizontal lines (top, under header, bottom — "bookshelf" layout).
- **PICO titles**: Specify Population, Intervention, Comparator, and Outcomes in every title.
- **No redundancy**: Tables must not duplicate data already in the text or figures.
- **Recasting**: If a table exceeds 2 pages, split into focused sub-tables or move to online supplement.
- **Information density**: Tabulate only data that reveals patterns or exact quantities beyond narrative.

## NEJM

**P-value precision (most specific guidance of any journal):**
- P > 0.01 → two decimal places (P=0.54)
- 0.001 ≤ P ≤ 0.01 → three decimal places (P=0.006)
- P < 0.001 → P<0.001 (never more decimal places)

**Units**: In column headers or leftmost column — not inside cells.

**Orientation**: Strictly portrait. No landscape tables.

**Abbreviations**: Defined in footnotes in alphabetical order.

**Odds/Hazard Ratios**: Report to two significant digits (e.g., 0.72, not 0.7213).

**Footnote symbols**: *, †, ‡, §, ¶ in order of appearance; then alphabetical for abbreviations.

## JAMA (AMA Style)

**P-values**: Exact values preferred (not just <0.05). Report to appropriate precision.

**Units**: Explicitly stated in column headers for every column.

**Orientation**: Portrait preferred; landscape with prior editorial approval only.

**Abbreviations**: Defined in footnotes, listed alphabetically.

**Data format**: Each value in its own individual cell — no manual spacing or tab-separated values.

**Multiplicity**: Clearly label P-values adjusted for multiple comparisons.

## The Lancet

**P-values**: Exact absolute values required. No "NS" — report exact P. Exception: P<0.0001 acceptable.

**Units**: SI units required. Drug names must use **rINN** (Recommended International Non-Proprietary Name).

**Decimal notation**: Midline decimal point — write `23·4` not `23.4` (Lancet house style).

**Data format**: Means + SD; medians + IQR (not ranges unless specified).

**Typography** (Lancet typesets — provide clean data):
- Table headings: 10pt Times New Roman **bold**
- Table legends: 10pt single-spaced
- Table data cells: 8pt single-spaced
- Headings within table: 8pt **bold**

**Orientation**: Portrait or landscape accepted (journal typesets); specify preference.

**Abbreviations**: Defined in the legend section below the table (not footnotes).

## The BMJ

**P-values**: Standardized reporting per BMJ statistical guidelines.

**Units**: SI units required throughout. **Exception**: Blood pressure must be in mmHg (not kPa).

**Orientation**: Portrait; maximum 2 printed pages. Longer tables → online supplementary material only.

**File format**: Word format required. Embedded Excel files are **rejected**.

**Heading hierarchy** (BMJ house style):
1. BOLD CAPS
2. **Bold lower case**
3. Plain text
4. *Italics*

**References**: Vancouver style — sequentially numbered in square brackets `[1]` or superscripts.

**Abbreviations**: Use sparingly; all must be defined in footnotes.

**Transparency**: Report absolute numbers before relative risks to enable NNH/NNT calculation.

**Open data**: BMJ encourages data availability statements linked to table data.

## Accessibility Requirements

- **Contrast ratio**: Minimum 4.5:1 for all text on colored backgrounds.
- **Shading**: Any cell shading must be explained in words in the footnotes — do not rely on color alone to convey information.
- **Screen readers**: Avoid merged cells where possible; use proper header markup in Word.
- **Alt text**: Required for tables submitted as images (though images are strongly discouraged).
