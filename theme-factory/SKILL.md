# Theme Factory

Apply consistent, professional styling to your artifacts.

## Available Themes

1. **Ocean Depths** - Professional and calming maritime
2. **Sunset Boulevard** - Warm and vibrant sunset colors
3. **Forest Canopy** - Natural and grounded earth tones
4. **Modern Minimalist** - Clean and contemporary grayscale
5. **Golden Hour** - Rich and warm autumnal palette
6. **Arctic Frost** - Cool and crisp winter-inspired
7. **Desert Rose** - Soft and sophisticated dusty tones
8. **Tech Innovation** - Bold and modern tech aesthetic
9. **Botanical Garden** - Fresh and organic garden colors
10. **Midnight Galaxy** - Dramatic and cosmic deep tones

## How to Use

1. **Show available themes** to the user
2. **Ask for their choice**
3. **Get explicit confirmation**
4. **Apply the theme** - colors and fonts throughout

## Theme Structure

Each theme includes:
- Cohesive color palette (hex codes)
- Font pairings (headings + body)
- Visual identity guidelines

## Application

### CSS Variables Example
```css
/* Ocean Depths Theme */
:root {
  --primary: #1a365d;
  --secondary: #2c5282;
  --accent: #63b3ed;
  --background: #f7fafc;
  --text: #1a202c;
  --heading-font: 'Playfair Display', serif;
  --body-font: 'Source Sans Pro', sans-serif;
}
```

## Custom Themes

Create custom themes when existing ones don't fit:
1. Name the theme (descriptive)
2. Define color palette
3. Choose font pairing
4. Apply consistently

## Use Cases

- Presentation slides
- Reports
- HTML landing pages
