# Meeting Insights Analyzer

This skill transforms meeting transcripts into actionable insights about communication patterns.

## When to Use This Skill

- Analyzing communication patterns across meetings
- Getting feedback on leadership and facilitation style
- Identifying conflict avoidance or indirect communication
- Understanding speaking habits and filler words
- Tracking improvement over time
- Preparing for performance reviews

## What This Skill Does

1. **Pattern Recognition**: Identifies recurring behaviors:
   - Conflict avoidance or indirect communication
   - Speaking ratios and turn-taking
   - Question-asking vs. statement-making
   - Active listening indicators
   - Decision-making approaches

2. **Communication Analysis**:
   - Clarity and directness
   - Use of filler words and hedging language
   - Tone and sentiment patterns
   - Meeting control and facilitation

3. **Actionable Feedback**: Provides specific examples with:
   - What happened
   - Why it matters
   - How to improve

4. **Trend Tracking**: Compares patterns over time

## How to Use

### Setup

1. Download meeting transcripts to a folder (e.g., `~/meetings/`)
2. Navigate to that folder
3. Ask for the analysis you want

### Examples

```
Analyze all meetings in this folder and tell me when I avoided conflict.
Look at my meetings from the past month and identify my communication patterns.
Give me feedback on my facilitation style in these meeting transcripts.
```

## Input Formats

- Text transcripts (.txt)
- VTT/SRT subtitle files
- JSON meeting exports
- Markdown notes

## Output

- Pattern analysis with examples
- Specific timestamps
- Actionable improvement suggestions
- Trends over time (if multiple meetings)
