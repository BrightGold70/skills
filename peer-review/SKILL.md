# Peer Review Skill

## Purpose

Conduct structured peer review of manuscripts using established criteria. Evaluate methodology, statistics, reporting quality, and provide constructive feedback.

## When to Use

- Reviewing a manuscript
- Preparing for submission
- Responding to reviewer comments
- Pre-submission self-check

---

## Review Framework

### Major Categories

1. **Originality & Significance**
2. **Methodology**
3. **Results & Analysis**
4. **Presentation**
5. **Ethics & Reporting**

---

## Detailed Checklist

### 1. Originality & Significance (Score: 1-5)

| Criterion | Questions |
|----------|-----------|
| Novelty | Does this add new knowledge? |
| Impact | Is the finding important? |
| Relevance | Who will benefit from this? |
| Literature | Adequately cited? |

### 2. Methodology

| Criterion | Questions |
|----------|-----------|
| Design | Appropriate study design? |
| Sample | Adequate sample size? |
| Controls | Proper control group? |
| Methods | Reproducible procedures? |
| Statistics | Correct statistical tests? |

### 3. Results

| Criterion | Questions |
|----------|-----------|
| Data | Data properly presented? |
| Figures | Clear and accurate? |
| Analysis | Appropriate methods? |
| Reproducibility | Enough detail to reproduce? |

### 4. Presentation

| Criterion | Questions |
|----------|-----------|
| Writing | Clear and concise? |
| Organization | Logical flow? |
| Figures | Publication quality? |
| Tables | Necessary and clear? |

### 5. Ethics

| Criterion | Questions |
|----------|-----------|
| Authorship | All contributors listed? |
| Conflicts | Conflicts disclosed? |
| Ethics | IRB/ethics approval stated? |
| Plagiarism | Original work? |

## Review Phrases

### Positive Feedback
- "The authors present a novel approach to..."
- "This study provides valuable insights into..."
- "The methodology is well-designed and..."
- "The results are clearly presented..."

### Constructive Criticism
- "The authors should consider..."
- "It would strengthen the paper if..."
- "This point would benefit from additional..."
- "The authors may want to clarify..."

### Requiring Action
- "The authors MUST address..."
- "Critical: Please provide..."
- "This requires revision before..."
- "Essential: Clarify..."

## Statistical Review Checklist

| Check | What to Look For |
|-------|------------------|
| Sample size | Power calculation mentioned? |
| Tests | Appropriate for data type? |
| Multiple testing | Correction applied? |
| Effect sizes | Reported, not just p-values? |
| Confidence intervals | Provided? |
| Normality | Tests for assumptions? |
| outliers | How handled? |
